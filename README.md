# 🎙️ AI Voice Studio Pro (IranVoiceTTS) V1.0

**استودیو هوشمند تبدیل متن به گفتار فارسی** یک ابزار قدرتمند و مدرن برای تولید صداهای باکیفیت انسانی از متون فارسی، طراحی شده با پایتون و رابط کاربری تخصصی.

---

## ✨ قابلیت‌های کلیدی (Key Features)

* **GUI مدرن و بهینه:** طراحی شده با کتابخانه `CustomTkinter` که ظاهری شبیه به اپلیکیشن‌های Native و صنعتی دارد.
* **تخصصی برای زبان فارسی:** بهینه‌سازی شده جهت پردازش صحیح کاراکترهای فارسی و استفاده از فونت استاندارد **Lalezar**.
* **مدیریت هوشمند تاریخچه (Surgical Updates):** لیست فایل‌ها بدون رفرش کل صفحه و به صورت جزیی به‌روزرسانی می‌شود تا سرعت برنامه در تعداد فایل بالا کم نشود.
* **سیستم پخش (Playback) پیشرفته:**
    * دارای پلیر داخلی با استفاده از `pygame`.
    * تشخیص خودکار اتمام صدا و بازگرداندن وضعیت دکمه‌ها (Polling System).
    * کنترل حجم صدا (Volume Control) به صورت لحظه‌ای.
* **عملیات فایل بدون واسطه:** تغییر نام (Rename) و حذف (Delete) فایل‌ها از داخل خود برنامه با دیالوگ‌های گرافیکی اختصاصی.
* **شخصی‌سازی بصری:** * پشتیبانی از **Dark Mode** و **Light Mode**.
    * قابلیت انتخاب لوگوی اختصاصی برای برندینگ شخصی.
* **جستجوی لحظه‌ای:** فیلتر کردن سریع صداهای ساخته شده بر اساس نام فایل.

## 🛠️ پیش‌نیازها و نصب (Installation)

برای اجرای این پروژه، باید پایتون ۳.۸ یا بالاتر داشته باشید. ابتدا پیش‌نیازها را نصب کنید:

```bash
pip install customtkinter pygame requests pillow
```

*نکته: برای نمایش صحیح متون، فایل فونت `Lalezar-Regular.ttf` باید در کنار فایل اصلی برنامه باشد.*

## 🚀 نحوه اجرا

ابتدا مخزن را کلون کنید و سپس فایل اصلی را اجرا کنید:

```bash
python voice_studio_pro.py
```

## ⚙️ پیکربندی (Configuration)

در فایل `voice_studio_pro.py` می‌توانید تنظیمات زیر را تغییر دهید:
* `VOICE_DIR`: پوشه ذخیره‌سازی صداها (پیش‌فرض: `voices`).
* `API_URL`: آدرس سرویس‌دهنده هوش مصنوعی. **(توجه: جهت استفاده رایگان از سرویس تبدیل متن به گفتار فارسی، از تغییر این آدرس خودداری کنید)**.

> ⚠️ **امنیت:** حتماً قبل از عمومی کردن کد، `API_TOKEN` خود را به یک متغیر محیطی (Environment Variable) منتقل کنید.

## 📸 محیط برنامه

* **ساخت صدا:** متن خود را وارد کنید و دکمه "ساخت صدا" را بزنید.
* **مدیریت:** از بخش History برای گوش دادن، تغییر نام یا حذف استفاده کنید.
* **تم:** با دکمه خورشید/ماه در هدر، ظاهر برنامه را تغییر دهید.
